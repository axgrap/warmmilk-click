API Examples

Open the browser console and try these commands:

  • client.getProjects() - Get all projects
  • client.getProject('example') - Get a specific project
  • client.getAllCommits() - Get all commits
  • client.searchCommits('image') - Search commits
  • project.getStats() - Get project statistics
  • project.getImages() - Get all images
  • image.getVersions() - Get image version history
